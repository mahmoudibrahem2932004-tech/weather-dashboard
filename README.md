# Weather Dashboard Pro

A comprehensive weather application with current conditions, 5-day forecast, temperature charts, and favorites management.

## Features
- Current weather display with temperature, humidity, wind speed, and cloud cover
- 5-day weather forecast with visual cards
- Interactive temperature chart using Chart.js
- Geo-location support
- Favorites management with localStorage
- Responsive design for all devices
- Error handling for invalid cities and network issues

## Technologies
- HTML5, CSS3, JavaScript (ES6+)
- Open-Meteo API (free, no API key required)
- Chart.js for data visualization
- LocalStorage for favorites persistence
- Geolocation API for location detection

## How to Use
1. Enter city name in search bar
2. Click Search or press Enter
3. Use "My Location" for current position
4. Add cities to favorites for quick access

## API Reference
- Geocoding: https://geocoding-api.open-meteo.com/v1/search
- Weather: https://api.open-meteo.com/v1/forecast
