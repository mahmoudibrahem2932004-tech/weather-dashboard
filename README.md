# Weather Dashboard Pro 🌤️

A weather app with current conditions, 5-day forecast, temperature charts, favorites, and Arabic city name support.

## 🚀 Live Demo
[https://weather-dashboard-2026.netlify.app](https://weather-dashboard-2026.netlify.app)

## ✨ Features
- Current weather: temperature, humidity, wind speed, cloud cover
- 5-day forecast with visual cards
- Temperature chart using Chart.js
- My Location button (geolocation)
- Favorites with localStorage
- Arabic city name support (automatically translates to English)

## 🛠️ Technologies
- HTML, CSS, JavaScript (ES6+)
- Open-Meteo API (free, no API key)
- Chart.js
- LocalStorage, Geolocation API

## 🔧 How to Use
1. Type a city name (English or Arabic)
2. Press Search or Enter
3. Use "My Location" for current position
4. Add cities to favorites

## 🌐 API Reference
- Geocoding: `https://geocoding-api.open-meteo.com/v1/search`
- Weather: `https://api.open-meteo.com/v1/forecast`
